## 📌 Project Title
**E-Commerce Customer Purchase Analysis Using Python, SQL, and Excel**

---

## 🎯 Objective
The main objective of this project is to analyze customer purchasing behavior in an e-commerce platform using data analysis techniques. It helps in identifying high-value customers, popular products, and revenue-generating cities.

---

## 🧠 Problem Statement
E-commerce platforms generate large amounts of customer transaction data, which is often difficult to analyze manually. Extracting meaningful insights such as customer spending patterns, product demand, and regional performance becomes challenging. This project addresses this problem by using data analysis tools to process and visualize the data effectively.

---

## 💡 Proposed Solution
This project uses SQL, Excel, and Python to analyze customer purchase data. The dataset is processed to calculate total spending, identify trends, and generate visual insights. The analysis helps businesses improve marketing strategies and decision-making.

---

## 🔑 Key Features
- Data loading from CSV file  
- Data preprocessing and cleaning  
- Creation of Total Purchase column  
- Customer spending analysis  
- Product popularity analysis  
- City-wise revenue analysis  
- Visualization using charts  
- Easy interpretation of business insights
